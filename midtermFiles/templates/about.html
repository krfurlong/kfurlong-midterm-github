Accountability: About This Data Set

Given the high degree of control exerted by the developers over the data collection process, the data presented in this web application is accountable and can be traced directly back to the source of the data. Using tools including the AWARE Framework App, Google Cloud, and R, each step along the data pipeline for this application is well-documented and transparent for interested parties.

Data Source: Where did this data come from?

This data was collected from one Apple iPhone 5s mobile device (Kyle Furlong's) on which the developers installed the AWARE Framework application directly from the Apple App Store. Information regarding the AWARE Framework and its various sensors can be found here. After downloading the application, the developers set up an instance in the Google Cloud in which they operated an auto-updating SQL database which periodically collected the most up-to-date data from the mobile application. Data for this project was collected near-continuously between February 15, 2018 at 9:47:11AM and March 5, 2018 at 7:48:52PM. Following the data collection period, it was then downloaded from the Google SQL database into a CSV format which the developers then imported into R for data cleaning. The location data was converted from CSV to GEOJSON using online open source applications, yet these sources should not have altered or jeopardized the quality or accountability of the data in any way, shape, or form.

Data Accessibility: Who has access to it?

The data as presented on this website is accessible only to Kyle Furlong, the author and developer of this site. During the course of the data collection and cleaning processes, it is possible that other entities or websites acquired parts or all of the data due to Terms and Conditions of usage. For example, the data was converted from its original CSV format to JSON and GEOJSON files through GeoJson.io and ConvertCSV.com which likely stored some of the data as "payment" for using their free services. However, though multiple entities may have access to the raw data, the data presented on the website can only be accessed and altered by the developer.
